交互说明

  • 散点图点击：点击学生圆点 → 右侧 Detail Panel 显示该学生详情（drag threshold 3px 防误触）
  • 散点图拖拽：鼠标按下拖动 ≥3px 触发平移，光标变为 grabbing；静止时光标为 default
  • 滚轮缩放：鼠标滚轮放大/缩小，范围 50%–400%
  • 双击象限：聚焦到对应象限区域（光标 zoom-in）
  • Detail Panel 姓名：点击姓名 → 打开 StudentModal
  • Detail Panel 按钮："查看学生详情 ↗" → 调用 onStudentClick(name)
  • 推荐卡片：点击推荐候选卡片 → 选中学生 + 打开 StudentModal
  • 深度互动排行：点击行 → 选中学生 + 打开 StudentModal

布局规格

  • 外层容器 .so-root：flex column，padding 20px 28px，gap 18px
  • Metrics 行：4 列等宽 grid，gap 10px
  • Matrix 行：1.55fr 1fr grid，gap 16px，最小高度 320px
  • Action 行：1.3fr 1fr grid，gap 16px
  • SVG viewBox：-10 -10 120 120（数据范围 0-100，留 10 单位边距）
  • 学生圆点半径：普通 2.5，选中 3.5（带 0.6px 描边）
  • 命名显示：每象限最多显示 1 个代表性学生名字 + 选中学生名字

颜色映射

  • 学优 (star)：var(--green) 圆点 / var(--green-soft) 背景
  • 努力但困惑 (struggling)：var(--amber) 圆点 / var(--amber-soft) 背景
  • 游刃有余 (coasting)：var(--blue) 圆点 / var(--blue-soft) 背景
  • 需要关注 (at-risk)：var(--red) 圆点 / var(--red-soft) 背景
  • AI 互动：var(--ai) 条形
  • 掌握度阈值：≥70% green, ≥40% amber, <40% red
  • 参与度阈值：≥60% green, ≥30% amber, <30% red

关键文件

  • SummaryOverlay.tsx — 组件主文件（散点图 + Detail Panel + 推荐 + 线索）
  • summary-helpers.ts — 数据计算（象限分类、知识点、候选推荐、AI 热度、深度排名）
  • teacher-right-panel.css — 所有 .so-* 样式定义
  • teacher-observe.css — .obs-stats-grid、.kp-* 共享样式
