Q

Quiz

"quiz"

多选一选择题。每题独立，O(answers.length) 同步评分。最基础的题型，新插件作者的模板参考。

同步 · No DI ✓ Migrated
M

Match

"match"

左右配对题。case-insensitive 字符串比较。共享选项池或题级选项。典型用法：单词配释义。

同步 · No DI ✓ Migrated
O

Order

"order"

序列排序题。0/100 二元评分。严格 Zod 校验：长度、范围、唯一性。典型用法：时间排序、流程步骤。

同步 · No DI ✓ Migrated

如何阅读
Lesson Bundle 适合教学设计师、课程编辑、销售人员。Plugin Bundle 适合插件作者、新加入团队的工程师。 从 lesson 进入：浏览 lesson → 看用到了哪些 plugin → 进入 plugin 看 schema。
