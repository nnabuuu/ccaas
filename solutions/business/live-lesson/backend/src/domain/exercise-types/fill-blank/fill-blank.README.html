Fill-blank
填空题

模板化句子里挖空，学生填词。accepts 数组列出可接受答案，前端先严格匹配，否则后端 LLM 模糊匹配（拼写/同义/形态）。支持多句子多空，per-blank 评分。

查看源码 → type: "fill-blank"
01 —

Answer Key Schema

const FillBlankBlankSchema = z.object({
  accepts: z.array(z.string()).nonempty(),  // 可接受答案（含同义/变体）
  hint:    z.string().optional(),
});

const FillBlankSentenceSchema = z.object({
  id:       z.string().min(1),
  template: z.string().min(1),       // 含 {{blankKey}} 占位
  blanks:   z.record(z.string(), FillBlankBlankSchema),
});

const FillBlankAnswerKeySchema = z.object({
  type:      z.literal('fill-blank'),
  sentences: z.array(FillBlankSentenceSchema).nonempty(),
});
02 —

字段说明

字段类型用途
sentences[].idstring句子唯一标识（用于结果定位）
sentences[].templatestring含 {{blankKey}} 占位的句子模板
sentences[].blanksRecord键为 blankKey，值为 { accepts, hint? }
blanks[k].acceptsstring[]可接受答案数组（含拼写变体/同义词）
blanks[k].hintstring?提示文本（答错时展示）
03 —

示例 Answer Key

{
  "type": "fill-blank",
  "sentences": [
    {
      "id": "s1",
      "template": "Ideas about beauty {{verb}} over time.",
      "blanks": {
        "verb": {
          "accepts": ["change", "changes", "vary", "differ"],
          "hint": "Look at ¶3 first sentence for the main verb."
        }
      }
    },
    {
      "id": "s2",
      "template": "The Venus of {{place}} is over {{age}} years old.",
      "blanks": {
        "place": { "accepts": ["Hohle Fels", "hohlefels"] },
        "age":   { "accepts": ["35000", "35,000"] }
      }
    }
  ]
}
04 —

Sanitize / Grade 行为

sanitize

剥离哪些字段

剥离 accepts（核心答案数据）。保留 template、id、hint（hint 仅在答错时展示，前期 sanitize 保留）。

// 学生看到
{
  type: "fill-blank",
  sentences: [
    { id: "s1", template: "Ideas {{verb}}..." }
  ]
}
grade

评分规则

两阶段：(1) 严格字符串匹配 accepts（lowercase trim）；(2) 未匹配则 LLM 判定是否同义/同形。每空独立 0/100 分。

// 输出
{
  total: 75,
  byDimension: {
    "s1.verb": true,
    "s2.place": true,
    "s2.age": false
  }
}
