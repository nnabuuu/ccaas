Stance
立场论证题

学生选 agree / disagree 并给出至少 N 条文本证据。2 维评分：立场正确性 + 证据数量。软完成模式（位置对+证据不足仍允许提交，total 50）。批判性阅读必备题型。

查看源码 → type: "stance"
01 —

Answer Key Schema

const StanceAnswerKeySchema = z.object({
  type:           z.literal('stance'),
  validPositions: z.array(z.string()).min(1),  // 可接受的立场，可单个或多个
  minEvidence:    z.number().int().min(1),       // 最少证据数
  stanceQ:        z.string().optional(),       // 立场问题（英）
  stanceQZh:      z.string().optional(),
  stanceOpts:     z.array(z.string()).min(2),  // 立场选项
  evidence:       z.array(z.string()).min(1),  // 学生可选的证据池
});
02 —

字段说明

字段类型用途
validPositionsstring[]所有可接受的立场（通常 ["agree"] 或 ["agree","disagree"] 都行）
minEvidencenumber最少证据条数，达不到 evidence 维度不得满分
stanceQ / stanceQZhstring?立场问题（如 "Is one beauty ideal more attractive?"）
stanceOptsstring[]立场单选选项（agree / disagree / neutral）
evidencestring[]课文中的可选证据条目（学生勾选）
03 —

示例 Answer Key

{
  "type": "stance",
  "stanceQ": "Is one idea of physical beauty really more attractive than another?",
  "stanceOpts": ["Agree", "Disagree", "It depends"],
  "validPositions": ["Disagree", "It depends"],  // 文本暗示 disagree 更合理
  "minEvidence": 2,
  "evidence": [
    "Ideas about physical beauty change over time (¶3)",
    "Venus of Hohle Fels is an overweight female (¶3)",
    "Egyptian art shows slim dark-haired women (¶3)",
    "Modern media bombards us with slim images (¶2)",
    "Some people question shallow beauty ideals (¶2)"
  ]
}
04 —

Sanitize / Grade 行为

sanitize

剥离哪些字段

剥离 validPositions 和 minEvidence。保留 stanceQ / stanceOpts / evidence。

// 学生看到
{
  type: "stance",
  stanceQ: "...",
  stanceOpts: [...],
  evidence: [...]
}
grade

评分规则

2 维评分：position 维度 +50（在 validPositions 中），evidence 维度 +50（数量 ≥ minEvidence）。位置对+证据不足 = 50 分软完成。

// 输出
{
  total: 100,  // 或 50, 0
  byDimension: {
    position: true,
    evidence: true
  }
}
