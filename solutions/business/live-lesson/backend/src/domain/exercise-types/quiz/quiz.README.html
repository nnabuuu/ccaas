Quiz
多选一选择题

最基础的选择题类型。每题独立，答对得满分，O(answers.length) 同步评分。所有题型中第一个迁移到 plugin 模式的样本，可作其他类型迁移参考。

Answer Key Schema Zod

服务端结构（含答案） 
// quiz.plugin.ts 内的 Zod schema
const QuizAnswerItemSchema = z.object({
  questionIdx:        z.number(),
  questionText:       z.string().min(1),
  questionTranslate:  z.string().optional(),
  options:            z.array(z.string()).min(2),
  correct:            z.number().int().nonnegative(),
  label:              z.string().optional(),
  hint:               z.string().optional(),
  hintZh:             z.string().optional(),
  walkthrough:        z.string().optional(),
  walkthroughZh:      z.string().optional(),
  paraRef:            z.array(z.number().int().positive()).optional(),
});

const QuizAnswerKeySchema = z.object({
  type:    z.literal('quiz'),
  answers: z.array(QuizAnswerItemSchema).nonempty(),
}).refine(
  (ak) => ak.answers.every((a) => a.correct < a.options.length),
  { message: 'quiz: correct index must be < options.length' },
);
02 —

字段说明 Fields

字段类型用途
type"quiz"固定字面量，用于 plugin 路由分发
answers[].questionIdxnumber题目顺序索引（从 0 开始）
answers[].questionTextstring题干文本（学生可见）
answers[].questionTranslatestring?题干翻译（双语阅读课用）
answers[].optionsstring[]选项数组，至少 2 个
answers[].correctnumber正确选项的索引（0-based，不是字母）
answers[].hintstring?答错时的提示（hintZh 中文版）
answers[].walkthroughstring?详细解答步骤（walkthroughZh 中文版）
answers[].paraRefnumber[]?关联段落号（点击高亮课文段落）

04 —

Sanitize / Grade 行为 Plugin Methods

sanitize

剥离哪些字段

从 answerKey 生成学生安全的 ExerciseSpec，剥离 correct / hint / walkthrough。保留 questionText / options / paraRef。

// sanitize 后学生看到的
{
  type: "quiz",
  questions: [
    { idx: 0, text: "...", options: [...] }
  ]
}
grade

评分规则

每题严格匹配：student.answers[idx] === correct 即正确。total = round(correct / total * 100)。byDimension 输出 { q0: true, q1: false, ... }。同步、纯函数、无 LLM。

// 输出
{
  total: 75,
  byDimension: { q0: true, q1: false, q2: true, q3: true }
}
