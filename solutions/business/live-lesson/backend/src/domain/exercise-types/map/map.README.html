Map
象限图题

二维 x/y 坐标系，学生把项目拖到合适位置。practice vs given：部分项目预先放好作示范，其余学生自己放。双重评分：位置接近度（欧氏距离）+ LLM 评 reasoning 文本质量。

查看源码 → type: "map"
01 —

Answer Key Schema

const MapAxisSchema = z.object({
  neg:   z.string().min(1),
  pos:   z.string().min(1),
  label: z.string().min(1),
});

const MapItemSchema = z.object({
  id:    z.string().min(1),
  label: z.string().min(1),
  hint:  z.string().optional(),
  refs:  z.array(z.number()).optional(),
});

const MapAnswerKeySchema = z.object({
  type:            z.literal('map'),
  prompt:          z.string().min(1),
  axes:            z.object({ x: MapAxisSchema, y: MapAxisSchema }),
  items:           z.array(MapItemSchema).min(1),
  expected:        z.record(z.string(), z.tuple([
    z.number().min(-1).max(1),
    z.number().min(-1).max(1),
  ])).optional(),  // item.id → [x, y] in [-1..1]
  minReasonLength: z.number().int().min(1).optional(),
  practiceCount:   z.number().int().min(1).optional(),
  randomPractice:  z.boolean().optional(),
});
02 —

字段说明

字段类型用途
axes.x / axes.y{neg, pos, label}两个轴的负/正端标签 + 轴名
items[]array所有可拖动的项目（id, label, hint?）
expectedRecorditem.id → [x, y] 坐标，范围 [-1, 1]
practiceCountnumber?前 N 个为练习项，后面是 given
randomPracticeboolean?随机选 practiceCount 个作为练习
minReasonLengthnumber?reasoning 文本最少字数（默认 8）

04 —

Sanitize / Grade 行为

sanitize

剥离哪些字段

剥离 expected。生成 givenPlacements：当 practiceCount < items 时，未练习的项预先放好（students 看见但不能动）。

// 学生看到（practiceCount=2）
{
  type: "map",
  mapItems: [...],  // 4 items
  givenPlacements: {
    "venus": { x: -0.3, y: -0.6 },
    "media-slim": { x: 0.8, y: 0.9 }
  }
  // fattening, egypt-art 学生自己放
}
grade

评分规则

双重评分：(1) 位置 — 学生坐标与 expected 的欧氏距离越小分越高；(2) reasoning — LLM 评 reasoning 文本质量。综合 score 加权平均。

// 输出
{
  total: 78,
  byDimension: {
    "fattening.pos": 90,
    "fattening.reason": 75,
    "egypt-art.pos": 80,
    "egypt-art.reason": 65
  }
}
