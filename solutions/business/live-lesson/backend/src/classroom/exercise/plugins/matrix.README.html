Matrix
矩阵填空题

多行多列结构化笔记。每行一个对象，每列一个维度。demo 行给学生看示例，practice 行让学生填。LLM 评分三维度（place / practice / reason）质量。cellQualities 返回每格 0-100 分。

查看源码 → type: "matrix"
01 —

Answer Key Schema

const MatrixAnswerItemSchema = z.object({
  rowIdx:     z.number(),
  place:      z.string().min(1),
  isDemo:     z.boolean().optional(),
  practice:   z.string().optional(),  // non-demo 必填
  reason:     z.string().optional(),  // non-demo 必填
  hint:       z.string().optional(),
  paraRef:    z.array(z.number().int().positive()).optional(),
  whatPrompt: z.string().optional(),  // "What practice did they have?"
  whyPrompt:  z.string().optional(),  // "Why did they have it?"
}).refine(
  (row) => row.isDemo || (row.practice && row.reason),
  { message: 'matrix: non-demo rows must have practice and reason' },
);

const MatrixAnswerKeySchema = z.object({
  type:          z.literal('matrix'),
  answers:       z.array(MatrixAnswerItemSchema).nonempty(),
  practiceCount: z.number().int().min(1).optional(),
});
02 —

字段说明

字段类型用途
answers[].rowIdxnumber行顺序索引
answers[].placestring第一列固定值（学生看，不填）
answers[].isDemoboolean?true = 示范行，practice/reason 给学生看；false = 学生要填
answers[].practicestring?正确答案 — 第二列（What/Practice）
answers[].reasonstring?正确答案 — 第三列（Why/Reason）
answers[].whatPromptstring?第二列的引导问题（如 "What practice?"）
answers[].whyPromptstring?第三列的引导问题（如 "Why this practice?"）

04 —

Sanitize / Grade 行为

sanitize

剥离哪些字段

demo 行保留 practice+reason（学生看示例）；non-demo 行只保留 place，剥离答案。保留 whatPrompt/whyPrompt 引导。

// 学生看到的 non-demo 行
{
  idx: 1,
  place: "Egypt",
  isDemo: false,
  whatPrompt: "What...?",
  whyPrompt: "Why...?"
}
grade

评分规则

跳过 demo 行。对每行 practice / reason 各调一次 LLM，返回 0-100 质量分。cellQualities 包含每格分数：{ rowIdx: { whatQ, whyQ } }。

// 输出
{
  total: 85,
  byDimension: {
    place: 100, practice: 80, reason: 75
  },
  cellQualities: {
    "1": { whatQ: 80, whyQ: 75 }
  }
}
