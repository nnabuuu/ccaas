# Fallback 策略：父节点 + 备注说明

## 问题场景

有些题目无法精确匹配到叶子节点，例如：

**题目**："用因式分解法解方程 x² + 5x + 6 = 0"

**问题**：题目只说"因式分解法"，但没有明确是哪种具体方法：
- 十字相乘法？
- 公式法（完全平方公式）？
- 提公因式法？

在候选池中展开"因式分解"后，看到10个子知识点，但无法确定选哪个。

## 解决方案：Fallback 到父节点 + 备注

### 核心规则

```
IF 无法精确匹配叶子节点 THEN
  1. 选择最近的父节点
  2. 降低置信度（0.6-0.8）
  3. 添加 note 字段说明：
     - 为什么无法精确匹配
     - 可能涉及哪些子知识点
     - 建议人工复核
END IF
```

## 置信度指南

| 置信度范围 | 场景 | 节点类型 | 说明 |
|-----------|------|---------|------|
| **0.95-1.0** | 精确匹配 | 叶子节点 | 题目明确使用该知识点 |
| **0.85-0.94** | 推理匹配 | 叶子节点 | 需要推理但很确定 |
| **0.70-0.84** | 暗示匹配 | 父节点 | 题目暗示但不明确 |
| **0.60-0.69** | 范围匹配 | 父节点 | 仅能确定大致范围 |
| **< 0.60** | 不建议 | - | 太不确定，不应标注 |

## 备注字段格式建议

### 标准格式

```
题目未明确[具体内容]。可能涉及的[子分类]：[列举选项]。建议人工复核确定[具体方面]。
```

### 示例模板

**因式分解未明确**：
```
题目未明确具体因式分解方法。可能涉及：十字相乘法、公式法（平方差/完全平方）、提公因式法。建议人工复核确定具体方法。
```

**不等式类型未明确**：
```
题目涉及不等式但未明确类型。可能是：一元一次不等式、一元二次不等式、不等式组。已标注为父节点。
```

**几何图形未明确**：
```
题目涉及三角形但未明确具体分类。可能涉及：直角三角形、等腰三角形、相似三角形等。建议根据题目条件确定。
```

**函数类型未明确**：
```
题目涉及函数但未明确具体类型（一次函数/二次函数/反比例函数）。已标注为父节点"函数"。
```

## 常见问题

### Q1: 何时使用父节点？

**A**: 满足以下任一条件：
- 题目使用笼统词汇（如"因式分解法"而非"平方差公式"）
- 候选池中有多个子节点都可能相关
- 从题目内容无法确定具体是哪个子知识点

### Q2: 置信度如何确定？

**A**: 根据匹配精确度：
- 精确到叶子节点 → 0.85-1.0
- 只能到父节点，但范围明确 → 0.7-0.84
- 只能到父节点，范围模糊 → 0.6-0.69

### Q3: note 字段必须写吗？

**A**:
- 选择父节点时 → **必须**写 note
- 选择叶子节点时 → 可选（通常不需要）

### Q4: 如果父节点还是不够精确？

**A**: 继续向上一级：
- Level 7 → Level 6 → Level 5
- 选择能覆盖题目范围的最低层级父节点
- 置信度相应降低

### Q5: 备注应该包含什么信息？

**A**: 必须包含：
1. **原因**：为什么无法精确匹配
2. **可能性**：列举可能涉及的子知识点（2-4个）
3. **建议**：提示人工复核或后续操作

## 总结

通过 **父节点 + 备注** 的 Fallback 策略：

1. ✅ **提高完整性**：即使无法精确匹配，也能标注到合理层级
2. ✅ **保持透明度**：note 字段说明了标注的局限性
3. ✅ **便于人工复核**：备注提供了可能的子知识点列表
4. ✅ **置信度合理**：降低置信度反映了匹配的不确定性

**核心原则**：
> "能精确就精确，无法精确就诚实说明" —— 标注质量 > 标注完美
