## 参考答案

### 一、选择题
1. B  2. C  3. B  4. A  5. B

### 二、填空题
6. 2x² 和 x²；-3x 和 -x（或：5 和 7）
7. 3a
8. 3x + 2x - y 或 5x - y
9. 3x - 2x + y 或 x + y
10. m = 2，n = 3
11. 3x - 4y

### 三、计算题
12. 原式 = (3x² - 5x²) + (2x + 4x) - 1
    = -2x² + 6x - 1

13. 原式 = 7a - 3b + 2a + 5b
    = (7a + 2a) + (-3b + 5b)
    = 9a + 2b

14. 原式 = 4m² - 3m + 1 - m² - 2m + 3
    = (4m² - m²) + (-3m - 2m) + (1 + 3)
    = 3m² - 5m + 4

15. 原式 = 2x + 6 - 3x + 6
    = (2x - 3x) + (6 + 6)
    = -x + 12

### 四、解答题
16. 原式 = 15a²b - 5ab² + 4ab² - 12a²b
        = (15a²b - 12a²b) + (-5ab² + 4ab²)
        = 3a²b - ab²

    当 a = 1，b = -1 时，
    原式 = 3×1²×(-1) - 1×(-1)²
        = -3 - 1
        = -4

17. （1）两次一共花费：
        (3x + 2y) + (2x + 3y)
        = 3x + 2y + 2x + 3y
        = (3x + 2x) + (2y + 3y)
        = 5x + 5y（元）

    （2）当 x = 5，y = 8 时，
        5x + 5y = 5×5 + 5×8
                = 25 + 40
                = 65（元）

        答：小明两次一共花了65元。

## 评分标准

**选择题**：每题4分，选对得满分，选错或不选不得分。

**填空题**：每题5分，答案完全正确得满分，部分正确酌情给分。

**计算题**：每题8分
- 去括号或移项正确：2分
- 找出同类项：2分
- 合并同类项过程：2分
- 最终结果正确：2分

**解答题**：每小题按步骤给分
- 化简过程正确：5分
- 代入求值正确：4分
- 应用题列式：4分，计算：3分，答：2分
