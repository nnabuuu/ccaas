KEY FEATURES
-----------------------------------------------------------------

✅ Two-Level Error Classification
   Level 1: 12 predefined error types (enum)
   Level 2: Natural language descriptions (Chinese)

✅ Error Types Supported:
   1. concept_misunderstanding    - 概念理解错误
   2. calculation_error           - 计算错误
   3. formula_misuse              - 公式应用错误
   4. step_omission               - 步骤遗漏
   5. step_order_wrong            - 步骤顺序错误
   6. condition_neglect           - 条件遗漏
   7. reasoning_error             - 推理错误
   8. symbol_confusion            - 符号混淆
   9. unit_conversion_error       - 单位换算错误
   10. range_error                - 取值范围错误
   11. special_case_neglect       - 特殊情况遗漏
   12. other                      - 其他

✅ 7-Step Analysis Workflow:
   Step 1: Get quiz information
   Step 2: Compare student vs standard answer
   Step 3: Classify errors (type + description + severity)
   Step 4: Identify affected knowledge points
   Step 5: Provide correct approach
   Step 6: Generate StudentAnswer object
   Step 7: Save to database (auto-aggregation)

✅ Integration with Error-Based Recommendation:
   - Auto-updates error_patterns table
   - Enables recommend_by_error_pattern tool
   - Powers teacher dashboard analytics
