## FAQ

**Q: How do I get a Free tier account?**

Free tier uses invite-code registration (no social login). Email [hello@kedgetech.io](mailto:hello@kedgetech.io) to request an invite code.

**Q: Who pays for LLM costs?**

- **Free:** Platform provides 50K tokens/month at no cost.
- **Starter / Business:** You bring your own LLM API key (Anthropic, OpenAI, etc.). You pay the model provider directly; the platform has no markup.
- **Demo track:** Platform covers LLM costs for the 60-day POC period.
- **Enterprise:** Client-owned infrastructure; your contract with the model provider.

**Q: Can I upgrade without migrating my data?**

Yes. All tiers run on the same platform. Upgrading is a plan change — no data migration, no re-deployment, no downtime.
