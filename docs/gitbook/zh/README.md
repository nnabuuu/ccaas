# 即见Agentic 是什么

即见Agentic 是一个 **Human-in-the-Loop AI 协作平台**，旨在弥合现代 AI 能力与企业遗留系统之间的鸿沟。

企业日常运营依赖大量遗留系统（SAP、Oracle、Salesforce、自定义应用），这些系统通常缺乏现代 API，自动化能力有限。当前的 AI 方案要么追求全自动化（风险高），要么仍需纯人工操作（效率低）。

即见Agentic 提供了第三条路径：**AI Agent 与人类操作员协同工作**，配备内置版本控制和审计追踪，确保每一步都可追溯、可回滚。

## 核心理念

- **人机协作**：AI 处理重复性任务，人类负责审核与决策
- **AI 引擎无关**：可接入任意 AI 后端（Claude Code、OpenAI、开源模型等）
- **遗留系统集成**：通过浏览器自动化、API、Webhook 连接现有系统
- **企业级治理**：多租户隔离、审计日志、版本控制
