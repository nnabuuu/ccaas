# 平台概述

LoopAI 是一个面向企业的 Human-in-the-Loop AI 协作平台，为 AI Agent 和人类操作员提供统一的实时协作工作空间。

## 平台定位

LoopAI **不是**一个 AI 模型或模型封装器，而是一个 **协作与集成平台**，提供：

- **版本控制** —— 每次变更（AI 或人类）都自动创建版本，附带完整审计追踪
- **人机协作** —— AI 输出实时同步到 Web 工作空间，人类可审核、编辑、优化
- **遗留系统集成** —— 无需现代 API 即可连接遗留系统
- **AI 灵活性** —— 可插拔的 AI 后端架构，不绑定任何特定 AI 供应商

## 平台组成

LoopAI 平台由以下核心组件构成：

| 组件 | 职责 |
|------|------|
| **Agent Engine** | AI Agent 执行引擎，管理会话生命周期 |
| **Skill System** | 技能路由与管理，支持关键词/模式/意图匹配 |
| **MCP Server** | 工具服务层，提供 AI Agent 可调用的外部工具 |
| **Solution Framework** | 面向垂直场景的完整应用框架 |
| **Vue SDK** | 前端集成 SDK，提供响应式状态管理 |
| **Admin Console** | 管理后台，提供租户管理、分析仪表盘 |

## 技术优势

1. **多租户架构** —— 租户级别的资源隔离与配额管理
2. **实时流式传输** —— 基于 WebSocket 的事件流，支持思考过程、工具使用、输出生成的实时展示
3. **Skill 路由** —— 基于触发器的智能路由，支持关键词、正则表达式、意图、上下文匹配
4. **MCP 集成** —— 标准化的工具协议，支持 REST API 适配和自定义工具开发
5. **消息持久化** —— 完整的对话历史存储，支持会话恢复
