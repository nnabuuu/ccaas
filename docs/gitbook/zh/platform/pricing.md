## 常见问题

**Q：如何获得 Free 账号？**

Free 方案采用邀请码注册（无社交登录）。请发送邮件至 [hello@kedgetech.io](mailto:hello@kedgetech.io) 申请邀请码，我们将在 1 个工作日内回复。

**Q：LLM 费用由谁承担？**

- **Free：** 平台每月提供 50K Token，用户无需自备 Key，零额外费用。
- **Starter / Business：** 用户自带 LLM API Key（支持 Anthropic、OpenAI 等），直接与模型提供商结算，平台不加价。
- **Demo 通道：** POC 期间（60 天）平台承担全部 LLM 费用。
- **Enterprise：** 甲方自有基础设施，自行与模型提供商签约。

**Q：升级方案需要迁移数据吗？**

不需要。所有方案运行在同一平台上。升级仅是套餐切换——无需数据迁移、无需重新部署、零停机时间。
