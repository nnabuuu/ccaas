## 关于 ADRs

架构决策记录 (Architecture Decision Records, ADR) 记录了影响软件架构的重要决策:

- **为什么做出这个决策?** (背景)
- **考虑了哪些替代方案?** (选项分析)
- **选择的方案是什么?** (决策)
- **有什么影响?** (后果)
