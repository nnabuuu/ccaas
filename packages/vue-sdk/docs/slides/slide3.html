Core Composables

useAgentState

Centralized agent state access via Vue inject. Processing status, tool activity, todos.

useFormBridge

Form registration with agent. Auto lifecycle management, data sync, submit handling.

useAIEditing

AI editing mode for section-based content. Progress tracking, callbacks, state machine.

usePlanMode

Plan proposal handling. Confirm/reject workflows, section preview, human-in-the-loop.

useTodoProgress

Todo tracking with computed helpers. Stats, progress percentage, filtering.

useToolActivity

Tool execution tracking. Current tool, history, duration, decision logic.
