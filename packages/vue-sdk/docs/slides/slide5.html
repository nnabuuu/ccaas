FormStateSynchronizer Service

Features

Centralized form state management

Field-level update tracking

Event pub/sub for cross-component sync

Old/new value diff for highlighting

Singleton or instance pattern

Usage

const sync = getFormStateSynchronizer() // Register form sync.registerForm('form-id', reactiveState) // Update from agent sync.updateFields('form-id', { title: 'New Title' }, 'agent') // Subscribe to updates sync.onFormUpdated((event) => { highlight(event.field) })
